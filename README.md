# dbx-dab-labs

The dab project was generated by using the default python template

## Getting Started

Before you run or deploy this project, you'll want to setup local Python environments for Databricks Connect and local PySpark development. Follow the instructions for your platform below

## Vitual Environment Setup

### macOS / Linux

1. Create and activate the Databricks Connect environment (using Python 3.11)
```
# at the project root
python3.11 -m venv .venv_dbc
source .venv_dbc/bin/activate
```
